Food Delivery Hackathon
Overview

This repository contains the analysis of a food delivery dataset, combining transactional, user, and restaurant data to study:

Order trends and user behavior

City-wise and cuisine-wise performance

Membership impact (Gold vs Regular)

Revenue distribution and seasonality

The project demonstrates real-world data merging and analysis using Python and Pandas.

Repository Contents
File	Description
Food_Delivery_Hackathon.ipynb	Jupyter Notebook containing all code, data merging steps, and hackathon answers
orders.xls	Transactional data of orders
users.json	User master data including membership information
restaurants.sql	Restaurant master data including cuisine and ratings
final_food_delivery_dataset.xls	Merged dataset generated by the notebook (optional)
README.md	This file, describing the project
How to Run

Clone the repository

git clone https://github.com/shravaninigudge20/Food_Delivery_Hackathon.git


Open the Jupyter Notebook

jupyter notebook Food_Delivery_Hackathon.ipynb


Run all cells from top to bottom

This will load the source files (orders.xls, users.json, restaurants.sql)

Merge the data

Generate final_food_delivery_dataset.xls (optional)

Compute all hackathon answers and analysis

Notes

The repository is public and can be viewed without logging in.

Ensure all source files are present in the same folder as the notebook.

The notebook is self-contained and reproduces all results.
